Manual Article Text Overrides

When a source is blocked as Source Check Required:

1. Open the article manually in your browser.
2. Copy the actual article body text.
3. Create a .txt file in this folder.
4. Name it close to the story title slug, or use the exact slug shown in the source-check prompt if available.
5. Paste the article text inside.
6. Run START_HERE.bat again.

If the pasted article text is strong enough, the reliability gate can upgrade it to publishable or usable-with-caution.

This is the fallback for sites that block automated extraction.
